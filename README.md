# 🐍 Python Learning Words
Обучающее веб-приложение для изучения терминологии Python. Проект является рандомайзером карточек по темам.
Благодаря гибкой системе редактирования словаря, приложение можно использовать не только для изучения синтаксиса программирования.

## 🚀 Основные функции
- **12 тематических разделов**: от основ синтаксиса до работы с ОС и статистикой.
- **RPG-система опыта и жизней**:
     ❤️ **Жизни**: 3 единицы на один тест.
     🔥 **Стаки**: бонусы к опыту за серию правильных ответов подряд.
     📈 **Прогрессия**: нелинейное усложнение прокачки после 10 уровня.
- **Умный словарь**: полноценная админ-панель для управления базой слов. 
   "Не понравился представленный перевод конструкции?" Не беда! Вы спокойно можете отредактировать как перевод, так и описание с самой конструкцией.
- **Кастомизация**: вы можете загрузить в `words.txt` любую базу данных (иностранные языки, даты и т.д.), придерживаясь формата.
- **Стабильность**: поддержка сложных ключей (фразы со знаками `?`, `,`, `/`) через URL-кодирование.
- **Динамический интерфейс**: приложение само строит меню на основе разделов в вашем `words.txt`.
- **Безопасность**: данные хранятся локально, внешние API не используются.
  
## 🛠 Технологии
- **Backend**: Python 3.12 + Flask
- **Frontend**: HTML5, CSS3 (Grid Layout, Animations)
- **Database**: Структурированный текстовый файл (.txt)

## 📦 Как запустить

### 1. Клонирование репозитория
```bash
git clone https://github.com/ChokotanMikaelFox/Python-learnig-words.git
cd Python-Learning-Words
```

### 2. Установка окружения
Убедитесь, что у вас установлен Python 3.10+. Установите Flask:
```bash
pip install Flask
```
*Или используйте виртуальное окружение:*
```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/MacOS:
source venv/bin/activate
pip install Flask
```

### 3. Запуск приложения
```bash
python Card.py
```

### 4. Доступ к тренажеру
- После запуска терминал покажет, что сервер активен.
- Откройте любой удобный браузер и перейдите по адресу: http://127.0.0.1:5000

**Важно:** Не закрывайте окно терминала во время работы — это "двигатель" веб-приложения.
